# Lead Generation Platform

Custom web application designed to capture, manage, and automate leads.

## Features
- Responsive landing page
- Admin dashboard for lead management
- Email notifications
- Telegram bot integration
- API & webhook automation

## Tech Stack
- PHP
- JavaScript
- MySQL
- Bootstrap

## Overview
This project demonstrates a full-stack solution combining frontend, backend, and real-time integrations to streamline lead management.
